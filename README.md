# Tic-tac-toe game

A implementation of the classic Tic-tac-toe game in bash.

## How to play
```
 1 | 2 | 3 
———+———+———
 4 | 5 | 6 
———+———+———
 7 | 8 | 9 
```

The grid is divided in nine numerated spaces. To select one just type in the number. You will be playing against the computer.
